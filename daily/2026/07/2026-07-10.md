## Rag Pipeline
- Data Indexing  & Vector Database
  - Dividing data in chunck
  - Then turn data into Vector
  - Storing in Vector Database
  
- Semantic  Data Retrival
  - Convert the user query into a vector.
  - Compare query vector with stored vector
  - Retrieve top‑k most relevant chunks/paragraphs.

- Augmentation & Generation
  - Context Assembly: Combine retrieved chunks with the original query into a single prompt.
  - Send this augmented prompt to the AI generator 
  - LLM produces a grounded response
